# 3D Визуализация Буквы - Лабораторные работы 6а-6с

## Описание проекта

Этот проект представляет собой приложение для визуализации трехмерной модели буквы с возможностью выполнения трехмерных преобразований и построения ортографических проекций. Проект реализует требования трех лабораторных работ по компьютерной графике.

## Функциональность

### Лабораторная работа 6а: Построение и визуализация трехмерных объектов
- Каркасная модель трехмерной буквы (по умолчанию - кириллическая 'А')
- Графический интерфейс с системой координат и осями
- Интерактивное вращение и масштабирование с помощью мыши
- Массив с координатами вершин для задания объекта

### Лабораторная работа 6b: Реализация трехмерных преобразований
- **Масштабирование** по осям X, Y, Z
- **Перенос** (трансляция) по всем трем осям
- **Вращение** вокруг осей X, Y, Z
- Отображение итоговой матрицы преобразования 4×4
- Возможность сброса преобразований

### Лабораторная работа 6с: Построение простейших проекций
- Три ортографические проекции на координатные плоскости:
  - Проекция на плоскость Oxy
  - Проекция на плоскость Oxz
  - Проекция на плоскость Oyz
- Автоматическое обновление проекций при изменении 3D модели
- Подписи осей на каждой проекции

## Технические требования

### Системные требования
- Операционная система: Windows/Linux/macOS
- Python версии: 3.7 или выше

### Зависимости
Для установки необходимых библиотек выполните:

```bash
pip install PyQt5 PyOpenGL numpy
```

## Установка и запуск

1. Клонируйте репозиторий или скачайте файлы проекта
2. Установите зависимости (см. выше)
3. Запустите приложение:

```bash
python3 main.py
```

## Использование

### Основное окно
1. **Левая часть окна**: 3D-вид модели
2. **Правая часть окна**: панель управления и проекции

### Управление 3D видом
- **Вращение**: левая кнопка мыши + перемещение
- **Вращение вокруг оси Z**: правая кнопка мыши + перемещение
- **Масштабирование**: колесико мыши
- **Переключение проекции**: выбор между перспективной и ортографической проекцией
- **Показ/скрытие осей**: флажок "Показать оси"

### Преобразования объекта
1. **Масштабирование**: задайте коэффициенты для осей X, Y, Z
2. **Перенос**: задайте смещения по осям
3. **Вращение**: задайте углы поворота в градусах
4. Нажмите "Применить преобразования"
5. Для сброса преобразований нажмите "Сбросить"

### Матрица преобразования
- В разделе "Матрица преобразования" отображается текущая матрица 4×4
- Матрица обновляется после каждого применения преобразований

### Ортографические проекции
- Проекции автоматически обновляются при изменении модели
- Каждая проекция подписана (XY, XZ, YZ)
- Оси координат обозначены соответствующими буквами

## Структура проекта

```
main.py     # Основной файл приложения
Letter3D                        # Класс для хранения 3D модели буквы
  |- vertices                   # Массив координат вершин
  |- edges                      # Список ребер (связей между вершинами)
  |- transform_matrix          # Матрица преобразования

GLWidget                        # Виджет OpenGL для 3D отображения
ProjectionWidget               # Виджет для ортографических проекций
MainWindow                     # Главное окно приложения
```

## Настройка модели буквы

Для изменения модели буквы отредактируйте класс `Letter3D`:

1. Измените массив `vertices` для вашей буквы
2. Настройте список `edges` для соединения вершин
3. Пример для другой буквы можно создать по аналогии с буквой 'А'

## Примечания

- Приложение использует матрицы преобразований 4×4 для поддержки однородных координат
- Все преобразования применяются последовательно и накапливаются
- Проекции строятся путем проецирования преобразованных координат на соответствующие плоскости
